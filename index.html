Introduccion a HTML

¿Qué es y cómo funciona?

HTML (o HyperText Markup Languaje) es un lenguaje de marcado que se utiliza para crear páginas web. HTML no es un lenguaje de programación, sino un lenguaje de marcado que se utiliza para estructurar el contenido de la página que estamos creando, además de darle sentido y significado a ese contenido.

Elementos, etiquetas y atributos

Un documento HTML está compuesto por elementos, que son las partes que componen una página web. Cada elemento tiene una etiqueta de apertura y una etiqueta de cierre. La etiqueta de apertura se escribe con el nombre del elemento entre corchetes angulares (<>) y la etiqueta de cierre se escribe con el nombre del elemento entre corchetes angulares y una barra (/) antes del nombre del elemento (). Por ejemplo, el elemento strong se escribe así: <strong> y su etiqueta de cierre es así: </strong>.

Aunque hay muchas etiquetas por defecto en HTML (y lo recomendable es usarlas siempre que se pueda) es posible crear elementos a partir de etiquetas con el nombre que queramos. Por ejemplo, un elemento <customTag>My own content</customTag> funcionaría de la misma forma que cualquier otro elemento, pero estamos perdiendo el significado que tienen las etiquetas por defecto para los motores de búsqueda.

Los elementos también pueden contener atributos.

Los atributos son propiedades que se le pueden asignar a un elemento. Por ejemplo, el elemento <a> tiene un atributo llamado href que se utiliza para indicar el destino del enlace. El elemento <img> tiene un atributo llamado src que se utiliza para indicar la ruta de la imagen que se va a mostrar.

Los atributos que establecen un valor siempre deben tener:

  • Un espacio entre ellos y el nombre del elemento (o entre el atributo previo si tiene más de un atributo).
  • Un signo igual (=).
  • El valor del atributo entre comillas dobles.

Estructura básica HTML (doctype, head, body...).

Cada documento HTML debe comenzar con el doctype, que es una declaración que indica al navegador que tipo de documento HTML se está utilizando. El doctype más utilizado es el HTML5, que se escribe así: <!DOCTYPE html>. La etiqueta doctype nos permite asegurarnos de que el navegador va a interpretar el documento HTML correctamente.

El documento HTML debe contener un elemento <html>. Este elemento envuelve todo el contenido de la página y es conocido como el elemento raíz. Este elemento también tiene atributos, siendo el más común el atributo lang, que nos permite especificar el lenguaje principal de nuestro documento.

Dentro del elemento <html> tendremos dos etiquetas principales: <head> y <body>. La etiqueta <head> contiene información sobre el documento, como el título de la página, los estilos, los scripts, etc. La etiqueta <body> contiene el contenido de la página.

El documento HTML debe contener un elemento <title> dentro de la etiqueta <head>. El elemento <title> es el título de la página y se muestra en la pestaña del navegador.

Otro elemento importante que se encuentran dentro del elemento <head> es el elemento <meta> que con ciertos atributos nos proporciona utilidades esenciales para cualquier pagina web. Por ejemplo, el elemento <meta charset="utf-8"> le dice al navegador que debe usar el formato de codificación UTF-8, el cual incluye los caracteres para la mayoría de lenguajes escritos, para representar el texto en nuestro documento. De la misma forma el elemento <meta name="viewport" content="width=device-width"> nos permite asegurarnos de que la página se renderizará como máximo en el ancho de la pantalla del dispositivo en donde se esta visualizando.

Texto y encabezados

Los elementos <h1> a <h6> se utilizan para definir los encabezados de la página. El elemento <h1> es el encabezado principal y el elemento <h6> es el encabezado de menor nivel. Los encabezados son importantes para la estructura de la página y para la accesibilidad.

Los elementos <p> se utilizan para definir párrafos de texto. Los párrafos son elementos de bloque, lo que significa que siempre comienzan en una nueva línea y ocupan todo el ancho disponible.

Los elementos <span> se utilizan para definir secciones de texto. Los elementos <span> son elementos de línea, lo que significa que no comienzan en una nueva línea y solo ocupan el ancho necesario.

Los elementos <strong> y <em> se utilizan para resaltar texto. El elemento <strong> indica que el texto es importante, mientras que el elemento <em> indica que el texto es enfatizado.

Los elementos <small> se utilizan para definir texto pequeño.

Los elementos <sub> y <sup> se utilizan para definir texto subíndice (H20) y texto superíndice (2256), respectivamente.

Los elementos <blockquote> y <q> se utilizan para definir bloques de citas y citas cortas, respectivamente. Por ejemplo:

"La unica forma de aprender un nuevo lenguaje de programación es escribiendo programas en él" - Dennis Ritchie

Los elementos <abbr> y <acronym> se utilizan para definir abbrs y acrónimos (HTML), respectivamente.

Los elementos <del> e <ins> se utilizan para definir texto eliminado y texto insertado, respectivamente.

Los elementos <pre> se utiliza para definir texto preformateado. Por ejemplo: 

fn main() {
    let mut count = 0u32;

    println!("Let's count until infinity!");

    // Infinite loop
    loop {
        count += 1;

        if count == 3 {
            println!("three");

            // Skip the rest of this iteration
            continue;
        }

        println!("{}", count);

        if count == 5 {
            println!("OK, that's enough");

            // Exit this loop
            break;
        }
    }
}

Los elementos <code> se utiliza para definir texto de código.

Listas

Los elementos <ul> y <ol> se utilizan para definir listas desordenadas y ordenadas, respectivamente. Los elementos <li> se utilizan para definir los elementos de la lista.

Los elementos <dl>, <dt> y <dd> se utilizan para definir listas de definiciones. El elemento <dt> se utiliza para definir el término de la definición, mientras que el elemento <dd> se utiliza para definir la definición.

Los elementos <ul> y <ol> pueden tener atributos type y value para definir el tipo de marcador y el valor de ese elemento en la lista, respectivamente.

Imagenes

Los elementos <img> se utilizan para definir imágenes. El elemento <img> tiene un atributo src que define la ruta de la imagen y un atributo alt en el que se define un texto alternativo para mostrar cuando no es posble mostrar la imagen o para permitir a los usuarios no pueden ver conocer el contenido de la imagen (los lectores de pantalla leen el texto en el atributo alt). 

<img src="path/to/image" alt="Texto alternativo">

Enlaces (links)

Los elementos <a> se utilizan para definir enlaces. El elemento <a> tiene un atributo href que define la ruta del enlace y un atributo target que define donde se abrirá el enlace. El atributo target puede tener los valores _blank (para abrir el enlace en una nueva pestaña) o _self (para abrir el enlace en la misma pestaña).

También se pueden crear enlances internos, es decir, que apuntan al contenido dentro de la misma página.

Introduccion a CSS

¿Qué es y cómo funciona?

CSS es un lenguaje de estilos que se utiliza para definir la apariencia de documentos HTML. CSS se utiliza para definir colores, fuentes, tamaños, márgenes, rellenos, etc. de elementos HTML.

CSS se utiliza para separar el contenido de la presentación. Esto significa que el contenido de una página web se define en HTML, mientras que la apariencia se define en CSS.

Vinculando el CSS con el HTML

Para vincular el CSS con el HTML, se utiliza la etiqueta <link> en el elemento <head> del HTML. El atributo href de la etiqueta <link> define la ruta del archivo CSS, mientras que el atributo rel define el tipo de relación entre el HTML y el CSS. 

<head>
...
    <link href="path/to/css" rel="stylesheet">
...
</head>

También se puede vincular el CSS con el HTML utilizando la etiqueta <style> en el elemento <head> del HTML. El contenido de la etiqueta <style> es el código CSS. 

<head>
    <style>
        /* Código CSS */
    </style>
</head>

También se pueden definir estilos en linea, es decir, en los atributos del elemento HTML. Aunque este forma de utilizar el código CSS es poco recomendada porque genera un HTML dificil de leer, depurar y mantener. 

<h1 style="color: red">Título</h1>

Selectores

Selector de tipo

El selector de tipo selecciona todos los elementos de un tipo específico. Por ejemplo, el selector h1 selecciona todos los elementos <h1>. 

h1 {
    color: red;
}

Selector de ID

El selector de ID selecciona un elemento con un ID específico. Por ejemplo, el selector #titulo selecciona el elemento con el ID titulo. 

#titulo {
    color: red;
}

Selector de clase

El selector de clase selecciona todos los elementos con una clase específica. Por ejemplo, el selector .titulo selecciona todos los elementos con la clase titulo. 

.titulo {
    color: red;
}

Selector de atributo

El selector de atributo selecciona todos los elementos que tienen un atributo específico. Por ejemplo, el selector [href] selecciona todos los elementos que tienen el atributo href. 

[href] {
    color: red;
}

Selector de pseudoclase

El selector de pseudoclase selecciona todos los elementos que cumplen una condición específica. Por ejemplo, el selector :hover selecciona todos los elementos que están siendo apuntados por el cursor del mouse. 

:hover {
    color: red;
}

Selector de pseudoelemento

El selector de pseudoelemento selecciona una parte específica de un elemento. Por ejemplo, el selector ::first-letter selecciona la primera letra de un elemento. 

::first-letter {
    color: red;
}

Otros selectores

Selector de atributo con valor

El selector de atributo con valor selecciona todos los elementos que tienen un atributo específico con un valor específico. Por ejemplo, el selector [href="https://www.google.com"] selecciona todos los elementos que tienen el atributo href con el valor https://www.google.com. 

[href="https://www.google.com"] {
    color: red;
}

Selector de atributo que contiene valor

El selector de atributo que contiene valor selecciona todos los elementos que tienen un atributo específico que contiene un valor específico. Por ejemplo, el selector [href*="google"] selecciona todos los elementos que tienen el atributo href que contiene el valor google. 

[href*="google"] {
    color: red;
}

Selector de atributo que inicia con valor

El selector de atributo que inicia con valor selecciona todos los elementos que tienen un atributo específico que inicia con un valor específico. Por ejemplo, el selector [href^="https"] selecciona todos los elementos que tienen el atributo href que inicia con el valor https. 

[href^="https"] {
    color: red;
}

Selector de atributo que termina con valor

El selector de atributo que termina con valor selecciona todos los elementos que tienen un atributo específico que termina con un valor específico. Por ejemplo, el selector [href$=".com"] selecciona todos los elementos que tienen el atributo href que termina con el valor .com. 

[href$=".com"] {
    color: red;
}

Selector de atributo que inicia con valor y termina con valor

El selector de atributo que inicia con valor y termina con valor selecciona todos los elementos que tienen un atributo específico que inicia con un valor específico y termina con un valor específico. Por ejemplo, el selector [href^="https"][href$=".com"] selecciona todos los elementos que tienen el atributo href que inicia con el valor https y termina con el valor .com. 

[href^="https"][href$=".com"] {
    color: red;
}

Selector de atributo que contiene valor y termina con valor

El selector de atributo que contiene valor y termina con valor selecciona todos los elementos que tienen un atributo específico que contiene un valor específico y termina con un valor específico. Por ejemplo, el selector [href*="google"][href$=".com"] selecciona todos los elementos que tienen el atributo href que contiene el valor google y termina con el valor .com. 

[href*="google"][href$=".com"] {
    color: red;
}

Propiedades en CSS

Una propiedad es una característica de un elemento que puede ser modificada. Por ejemplo, el color de un elemento puede ser modificado utilizando la propiedad color. Una propiedad puede tener uno o más valores. Por ejemplo, la propiedad color puede tener el valor red o blue.

Propiedades de texto

Las propiedades de texto modifican el texto de un elemento. Por ejemplo, la propiedad color modifica el color del texto de un elemento. Las propiedades de texto son:

  • color
  • font-family
  • font-size
  • font-style
  • font-weight
  • letter-spacing
  • line-height
  • text-align
  • text-decoration
  • text-indent
  • text-transform
  • word-spacing

Propiedades de fondo

Las propiedades de fondo modifican el fondo de un elemento. Por ejemplo, la propiedad background-color modifica el color de fondo de un elemento. Las propiedades de fondo son:

  • background-color
  • background-image
  • background-position
  • background-repeat
  • background-size

Animaciones

Las animaciones son una forma de modificar las propiedades de un elemento de forma gradual. Por ejemplo, puedes animar el color de fondo de un elemento de forma gradual utilizando la propiedad background-color. Un ejemplo de animacion sencilla es: 

div {
    background-color: red;
    transition: background-color 2s;
}

div:hover {
    background-color: blue;
}

En el ejemplo anterior, el color de fondo de un elemento div cambia de rojo a azul en 2 segundos cuando el mouse pasa sobre el elemento.

Nota: CSS es un lenguaje de programación, por lo que puedes crear animaciones muy complejas. Aqui puedes encontrar más información sobre animaciones en CSS.

Otras propiedades

Propiedades de bordes

Las propiedades de bordes modifican los bordes de un elemento. Por ejemplo, la propiedad border-color modifica el color de los bordes de un elemento. Las propiedades de bordes son:

  • border-color
  • border-style
  • border-width

Propiedades de dimensiones

Las propiedades de dimensiones modifican las dimensiones de un elemento. Por ejemplo, la propiedad width modifica el ancho de un elemento. Las propiedades de dimensiones son:

  • height
  • width

Propiedades de posicionamiento

Las propiedades de posicionamiento modifican la posición de un elemento. Por ejemplo, la propiedad position modifica la posición de un elemento. Las propiedades de posicionamiento son:

  • bottom
  • left
  • position
  • right
  • top

Propiedades de márgenes

Las propiedades de márgenes modifican los márgenes de un elemento. Por ejemplo, la propiedad margin-top modifica el margen superior de un elemento. Las propiedades de márgenes son:

  • margin-bottom
  • margin-left
  • margin-right
  • margin-top

Buenas prácticas

Buenas prácticas para HTML:

  • Utiliza etiquetas HTML semánticas para describir el contenido de tu página, en lugar de utilizar etiquetas solo para fines de formateo. Por ejemplo, utiliza <h1> para encabezados y <p> para párrafos, en lugar de utilizar etiquetas <font> o <div>.
  • Utiliza etiquetas y nombres de atributos en minúsculas.
  • Utiliza comillas alrededor de los valores de atributo.
  • Utiliza un elemento <label> para cada control de formulario.
  • Utiliza texto alternativo para las imágenes.
  • Utiliza elementos estructurales de HTML5 (por ejemplo, <header>, <footer>, <nav>, etc.) para organizar tu contenido.

Buenas prácticas para CSS:

  • Utiliza un estilo consistente en todo tu sitio web.
  • Utiliza un archivo CSS separado para los estilos, en lugar de utilizar estilos inline.
  • Utiliza una hoja de estilos de reset o normalize para garantizar estilos consistentes en diferentes navegadores.
  • Utiliza nombres de clase semánticos que describan el propósito del elemento, en lugar de utilizar nombres de clase presentacionales.
  • Utiliza los selectores más cortos y eficientes posibles.
  • Utiliza el selector menos específico necesario para dar estilo a un elemento.
  • Utiliza la cascada a tu favor, en lugar de luchar en contra de ella.
  • Utiliza técnicas de diseño responsive basadas en móviles.
