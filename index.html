Introduccion a HTML

¿Qué es y cómo funciona?

HTML (o HyperText Markup Languaje) es un lenguaje de marcado que se utiliza para crear páginas web. HTML no es un lenguaje de programación, sino un lenguaje de marcado que se utiliza para estructurar el contenido de la página que estamos creando, además de darle sentido y significado a ese contenido.

Elementos, etiquetas y atributos

Un documento HTML está compuesto por elementos, que son las partes que componen una página web. Cada elemento tiene una etiqueta de apertura y una etiqueta de cierre. La etiqueta de apertura se escribe con el nombre del elemento entre corchetes angulares (<>) y la etiqueta de cierre se escribe con el nombre del elemento entre corchetes angulares y una barra (/) antes del nombre del elemento (). Por ejemplo, el elemento strong se escribe así: <strong> y su etiqueta de cierre es así: </strong>.

Aunque hay muchas etiquetas por defecto en HTML (y lo recomendable es usarlas siempre que se pueda) es posible crear elementos a partir de etiquetas con el nombre que queramos. Por ejemplo, un elemento <customTag>My own content</customTag> funcionaría de la misma forma que cualquier otro elemento, pero estamos perdiendo el significado que tienen las etiquetas por defecto para los motores de búsqueda.

Los elementos también pueden contener atributos.

Los atributos son propiedades que se le pueden asignar a un elemento. Por ejemplo, el elemento <a> tiene un atributo llamado href que se utiliza para indicar el destino del enlace. El elemento <img> tiene un atributo llamado src que se utiliza para indicar la ruta de la imagen que se va a mostrar.
